SmartPing

一款开源、高效、便捷的网络质量监控神器！


功能

支持正/反向PING绘图与时间查询，支持组网后的拓扑显示，支持报警规则设定。


特性

Go语言编写，跨平台；去中心化设计，数据自动周期性清理免维护.

开源

代码100%开源并可无条件免费使用。所有的源代码均通过 Apache 授权协议并托管在 GitHub 上。

跨平台

任何 Go 语言 支持的平台都可以运行 SmartPing，包括 Windows、Mac、Linux 以及 ARM。

无中心化

独特的 去中心化、数据循环 写入设计，并采用嵌入式数据库 Sqlite 进行数据存数。
